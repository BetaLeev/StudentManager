
# 教务管理系统 

## 前端 Vue , 后端 koa2


###安装
客户端：
                    

1 `cd client `

2 `npm install` 或者 `yarn`

3 `npm run dev` 或者 `yarn serve`

服务器端

1 `cd server`

2 `npm install`

3 `npm start`
